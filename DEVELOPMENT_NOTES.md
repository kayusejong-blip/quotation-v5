# 📝 자동 견적서 프로그램 개발 노트

현재 기준(**2026-07-14**) 프로그램의 개발 현황 및 업데이트 로그입니다.

## 🚀 현재 버전: **v5.8**

---

### 📅 최근 업데이트 로그

#### **v5.8** (최신)
- **자주 사용하는 품목(헬스매트) 버튼 UI 최적화**:
  - 버튼 텍스트에 묶음 수량(4p, 3p)을 명시하여 직관성 확보
  - 입력란 길이를 줄이고 버튼 너비를 확장하여 텍스트가 1줄로 깔끔하게 표시되도록 개선

#### **v5.7**
- **개별 항목 할인율 적용 기능 추가**:
  - 기존 일괄 할인율 방식을 개별 적용 방식으로 개편
  - 견적서 테이블에 '할인(%)' 열 추가, 배송비 등 할인 미적용 항목 설정 가능

#### **v5.6**
- **자주 사용하는 품목 단가 입력**:
  - 기존 헬스매트 20t, 25t 고정 단가 추가 기능에 **단가 직접 입력 필드** 추가
  - 유동적인 판매 가격에 맞춰 사용자가 입력한 단가로 견적서 항목이 추가되도록 개선

#### **v5.4**
- **설정 기능 강화**: 할인율 직접 입력 칸 옆에 '적용' 버튼 추가
  - 입력 후 명시적으로 적용 버튼을 누를 때 합계 금액이 재계산되도록 UI/UX 보완
  - 적용 시 "대장님, 할인율이 적용되었습니다!" 알림 메시지 출력

#### **v5.3**
- **입력 폼 최적화**: 견적서 작성 탭에서 불필요한 '가로/세로 매트 입력(m)' 필드 제거
- **레이아웃 고도화**: 제품명 및 옵션 입력 칸의 너비 조정 및 모바일 대응 강화

#### **v5.1**
- **수량 계산기 업그레이드**:
  - 계산기 UI 레이아웃 전면 개편
  - 주요 제품 규격(30×30, 50×50, 1m×1m) 프리셋 버튼 추가
  - 계산된 수량을 클릭 한 번으로 견적서에 즉시 반영하는 기능 추가

#### **v5.0** (대규모 업데이트)
- **멀티 탭 인터페이스**: '견적서 작성'과 '수량 계산기'를 탭으로 분리
- **반응형 디자인**: 사이드바 기반의 프리미엄 관리자 패널 UI 도입
- **직인(Stamp) 시스템**: 대표자 날인 이미지(`stamp.png`) 자동 렌더링 지원

---

### 🛠️ 주요 탑재 기능

1. **문서 내보내기**
   - **📄 PDF 저장**: 전체 견적서를 고해상도 PDF 파일로 저장
   - **🖼️ 이미지(PNG) 저장**: 견적 영역을 깔끔한 이미지로 캡처하여 저장
   - **📋 클립보드 복사**: 이미지 형태로 클립보드에 즉시 복사 (카톡 등 전송 용이)

2. **스마트 계산 및 관리**
   - **📐 자동 수량 계산기**: 시공 공간 면적 대비 필요 제품 수량 산출
   - **💾 로컬 스토리지 저장**: 작성 중인 견적 데이터를 브라우저에 영구 저장 및 불러오기
   - **⭐ 즐겨찾기**: 자주 찾는 견적 내역 별도 관리

3. **커스터마이징**
   - **할인율 적용**: 전체 금액에 대한 퍼센트(%) 할인 자동 계산
   - **실시간 편집**: 상품명, 규격, 단가, 수량 등 모든 필드를 클릭하여 즉시 수정
